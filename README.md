Sistema de Gestión de Metas Personales

Descripción:
Proyecto final de Programación 3 en ASP.NET Core MVC con SQL Server. Permite crear, editar, eliminar y visualizar metas y tareas, asignando prioridad, estado y fechas límite.

Tecnologías:

ASP.NET Core MVC

C#

SQL Server + Entity Framework Core

Bootstrap 5

Funcionalidades principales:

Metas: CRUD, categoría, prioridad, estado, fecha límite

Tareas: CRUD, asociadas a metas, estado, dificultad y tiempo estimado


Ejecución:

Clonar repositorio

Configurar cadena de conexión en appsettings.json

Ejecutar migraciones: Update-Database

Ejecutar proyecto en Visual Studio
